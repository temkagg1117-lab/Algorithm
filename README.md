🐪 Escape Game – Элсэн цөлөөс оргож зугтах

Энэ бол Python хэл дээр бичигдсэн текстэн (console) адал явдалт тоглоом юм. Та элсэн цөл дунд байгаа шоронгоос оргож, тэмээгээр зугтаж, цагдаа нараас мултарч нуувчиндаа хүрэх ёстой.

🎮 Тоглоомын зорилго
200 км зам туулж нуувчинд хүрэх
Цагдаа нараас зугтаж амьд үлдэх
Цангаа болон тэмээний ядаргааг зөв удирдах
⚙️ Ашигласан технологи
Python 3
random модуль
🕹️ Хэрхэн тоглох вэ
